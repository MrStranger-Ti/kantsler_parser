Данная программа служит для парсинга XML файла с сайта kantsler24.ru

------
Запуск
------

Перед использованием необходимо указать ссылку на XML файл в конфигурационном файле (config.ini).
Для запуска программы запустите исполняемый файл parser.exe.

Если вдруг возникла ошибка или программа работает неправильно, то:

- проверьте корректность настроек в конфигурационном файле config.ini;
- проверьте логи программы, они находятся в папке logs. Они могут помочь решить проблему.

-----------------
Файл конфигурации
-----------------

В конфигурационном файле есть несколько настроек, обязательные (required) и необязательные (optional).

Обязательные настройки
----------------------
URL - ссылка на xml файл

Необязательные настройки
------------------------
NONAME_BRAND_VALUE - как будет указываться бренд в Excel, если он не будет найден в названии товара
                     по умолчанию: Noname

RESULT_FILE_NAME - название готового Excel файла
                   по умолчанию: result.xlsx

RESULT_FILE_DIR - папка, куда будет сохраняться готовый Excel файл
                  по умолчанию: в корневую папку, в которой находится parser.exe

SHEET_NAME - название листа в шаблонном Excel файле
             по умолчанию: Ассортимент

----------------------
Текстовый файл брендов
----------------------

В файле brands.txt перечисляются бренды, которые используются для поиска в названии продукта.
Каждый бренд должен быть на отдельной строке.
