Данная программа служит для получения и обработки данных с api сайта api.samsonopt.ru

------
Запуск
------

Перед использованием необходимо указать все обязательные настройки в конфигурационном файле (config.ini).
Для запуска программы запустите исполняемый файл main.exe.

Если вдруг возникла ошибка или программа работает неправильно, то:

- проверьте корректность настроек в конфигурационном файле config.ini;
- проверьте логи программы, они находятся в папке logs. Они могут помочь решить проблему.

-----------------
Файл конфигурации
-----------------

В конфигурационном файле есть несколько настроек, обязательные (required) и необязательные (optional).

Обязательные настройки
----------------------
API_KEY - ключ API

Необязательные настройки
------------------------
RESULT_FILE_NAME - название готового Excel файла
                   по умолчанию: result.xlsx

RESULT_FILE_DIR - папка, куда будет сохраняться готовый Excel файл
                  по умолчанию: в корневую папку, в которой находится main.exe

SHEET_NAME - название листа в шаблонном Excel файле
             по умолчанию: Ассортимент

PAGINATION_PAGES - Количество страниц товаров
                   По умолчанию: 5

PAGINATION_COUNT - Количество товаров на странице
                   Примечание: Если количество товаров будет превышать 50000, то api выдаст ошибку
                   По умолчанию: 10000

TIMEOUT - Время ожидания ответа от сервера в секундах
          Примечание: если PAGINATION_COUNT будет слишком большим, то время на обработку увеличится
          По умолчанию: 10
